# Discord 表情反應標記機器人

掃描 Discord 訊息上的所有表情反應，自動 @mention 每個反應過的用戶（跨表情去重），並支援附加自訂訊息。

---

## 功能

- 掃描指定訊息的所有表情符號反應
- 自動標記（@mention）反應用戶，跨表情去重（每人只被 @mention 一次）
- 支援附加自訂訊息，一併送出
- 自動過濾機器人的反應
- 防止意外觸發 @everyone / @here

---

## 所需權限

| 權限             | 說明             |
| ---------------- | ---------------- |
| 檢視頻道         | 存取目標頻道     |
| 讀取訊息歷史記錄 | 抓取訊息及其反應 |
| 傳送訊息         | 回覆結果         |
| 嵌入連結         | 顯示訊息跳轉連結 |

---

## 使用方式

| 指令                          | 功能                         |
| ----------------------------- | ---------------------------- |
| 回覆訊息 → `!reactions`       | 讀取該訊息的所有反應         |
| 回覆訊息 → `!reactions 文字`  | 讀取反應並附加自訂訊息一併送出 |

### 輸出範例

```
活動開始囉！
@小明 @小美 @阿哲 @大衛 @小芳 @阿偉
```
